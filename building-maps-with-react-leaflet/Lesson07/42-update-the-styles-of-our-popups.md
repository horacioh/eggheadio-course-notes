# Update the Styles of our Popups

**[ðŸ“¹ Video](https://egghead.io/lessons/egghead-update-the-styles-of-our-popups)**

Our popups can be styled just like any other HTML element. Let's start by adding classes and then add styling to the `App.css` file.

Below is the example styling by Colby.

```css
.restaurant-popup h3 {
  font-size: 1.4em;
  margin-bottom: 0.4em;
}

.restaurant-popup ul {
  padding: 0;
  list-style: none;
  margin: 0;
}

.restaurant-popup li {
  margin-bottom: 0.4em;
}

.restaurant-popup li:last-child {
  margin-bottom: 0;
}
```
